## Image Generation
When user asks to generate/create/draw images, **you MUST call `generate_image` tool**.
**DO NOT** draw ASCII art, describe 3D workflows, or generate images yourself.
**Immediately call the tool** — the image will auto-display in chat.
See the tool entry below for parameters.

### Supported Providers
- SD WebUI (local, e.g. http://127.0.0.1:8200)
- ComfyUI (auto-detect, e.g. http://127.0.0.1:8200)
- OpenAI DALL-E
- Replicate (SDXL, Flux, etc.)
- Pollinations AI (free, no API key)
