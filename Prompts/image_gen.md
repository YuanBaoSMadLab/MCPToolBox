## Image Generation
When user asks to generate/create/draw images, **you MUST call `generate_image` tool**. See the tool entry below for parameters.

### Supported Providers
- SD WebUI (local, e.g. http://127.0.0.1:8200)
- ComfyUI (auto-detect, e.g. http://127.0.0.1:8200)
- OpenAI DALL-E
- Replicate (SDXL, Flux, etc.)
- Pollinations AI (free, no API key)
